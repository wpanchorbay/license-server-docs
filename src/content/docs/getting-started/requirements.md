---
title: Requirements
description: Review the runtime and development requirements for License Server.
---

## Runtime requirements

- WordPress 6.8 or newer
- PHP 7.4 or newer
- WooCommerce 9.1 or newer

## Development requirements

- Bun or Node.js for the admin frontend build
- Composer for PHP dependencies

## Plugin prerequisites

License Server assumes WooCommerce is active and the site is intended to sell digital software products such as:

- WordPress plugins
- WordPress themes
- desktop software
- other license-key-based digital goods
